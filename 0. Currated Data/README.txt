This folder includes data and files on the Data Curation section, including:
+ BindingDB Database Curation: raw downloaded file, data cleaning/processing code and log, full cleaned dataset, data distribution, and outlier check (code for data distribution plot and outlier check plot are in the LBDD QSAR Colab Notebook).
+ Protein Structure curation, including raw PDB downloaded from RCSB (ID: 4XXS), processed version, H++ protonation results, and the final PDBQT version.
+ Fragment libraries for both LBDD and SBDD pathways, including raw datasets, code for selecting fragments, etc.
